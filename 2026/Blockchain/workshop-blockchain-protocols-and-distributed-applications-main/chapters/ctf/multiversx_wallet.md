# MultiversX Wallet & Transfer

Please use the devnet network for Multivers:
* [Wallet](https://devnet-wallet.multiversx.com/)
* [Explorer](https://devnet-explorer.multiversx.com/)
* [Exchange](https://devnet.xexchange.com/)

## Token Transfer(10p)

**Objective:** You need to create a wallet on MultiversX blockchain, get some xEGLD (testnet EGLD tokens) and make a transfer to our account: `erd1mqa9wttlzwwdvwgk9dzsfdn79lv5raw0tfe9ynvn0dg92hpruvaqhhd2gx`

You can check the [MultiversX Wallet](https://cs-pub-ro.github.io/workshop-blockchain-protocols-and-distributed-applications/Practical%20Sessions/MultiversX%20Wallet/) section.

## Token Creation (10p)

## NFT Creation (10p)

**Objective:** You need to create a new  NFT token on MultiversX blockchain, and make a transfer of that NFT to our account: `erd1mqa9wttlzwwdvwgk9dzsfdn79lv5raw0tfe9ynvn0dg92hpruvaqhhd2gx`.
